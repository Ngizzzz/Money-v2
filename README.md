# Money — PWA Pencatat Keuangan

Aplikasi pencatatan pemasukan & pengeluaran berbasis Progressive Web App (PWA).
Bisa diinstall di HP dan desktop, bekerja offline, sync ke Google Sheets.

## Fitur
- Catat pemasukan & pengeluaran dengan kategori
- Tema Dark & Light
- Sync otomatis ke Google Sheets (via API Key)
- Export CSV
- Laporan grafik bulanan
- Bisa diinstall sebagai aplikasi (PWA)
- Bekerja offline

Lihat panduan lengkap di README ini atau ikuti tutorial deploy ke Vercel.
